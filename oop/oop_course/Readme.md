# OOP Concepts and Notes
Main problem that is solved by OOPs is   
1. Define your own datatype and its behaviour.   
2. From General to specific problem solving.   
### Topics for OOPs
1. Classes     
Classes are self defined datatypes a.k.a Blue print for the creation of object. It is also self defined data type. **All datatypes are built-in classes in python.** A class always has two important component   
a. data/attribute/Property    
b. Function/methods for its behaviour      
2. Objects    
Object is the instance of the class. Real memory allocation happens here.
